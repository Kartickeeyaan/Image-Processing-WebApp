Learn how to use the CV Ops image processing tool and get the most out of it.

Getting Started

CV Ops allows you to easily upload images and apply various image processing operations. Follow these simple steps:

  1. Choose a category from the dropdown (e.g., "Filters", "Transformation").
  2. Select an operation (e.g., "Mean Filter", "Gamma Transformation").
  3. Upload the image you'd like to process.
  4. Click the "Apply Operation" button to view the processed result.

Supported Image Operations

Here’s a list of operations you can apply to your image:

  • Gamma Transformation — Adjust the brightness and contrast.
  • Log Transformation — Apply logarithmic scaling for enhancing darker regions.
  • Thresholding — Convert images to binary based on a set threshold.
  • Filters — Apply different filters such as Gaussian, Sobel, etc.

Common Issues and Troubleshooting

  • Issue: Image is not uploading
    Ensure the image is either a JPG or PNG format and does not exceed the size limit.
  • Issue: Operations not showing
    Make sure you've selected a category first.
